
# Legacy Guardians: 传承守护者 - 财富之旅

## 📜 项目概览 (Project Overview)

`Legacy Guardians`（传承守护者）是一个旨在教育青少年关于基础金融知识和投资原理的 Web 模拟游戏。玩家将通过三个循序渐进的关卡，从银行储蓄开始，逐步过渡到股票交易和多元化投资组合管理。

**游戏目标：** 玩家通过学习和实践不同的投资策略，在模拟环境中实现财富的稳健增长。

**核心特色：**

- **分级教学：** 从低风险的银行储蓄到高阶的投资组合管理，难度逐步提升。
- **AI 投资教练：** 在第三关引入 AI 聊天功能，为玩家提供个性化的投资建议和指导（基于 OpenAI）。
- **前后端分离：** 后端由 Python Flask 驱动，管理游戏状态和逻辑；前端使用 HTML/CSS/JavaScript 进行渲染。

- 

| **领域** | **技术**                | **描述**                                                     |
| -------- | ----------------------- | ------------------------------------------------------------ |
| **后端** | Python 3.x, Flask       | RESTful API，处理游戏逻辑和状态管理。                        |
| **数据** | Pandas                  | 用于加载、处理和访问市场价格 (`mock_market_prices.csv`)、资产信息 (`mock_assets.csv`) 和任务信息 (`missions_catalog.csv`) 等数据。 |
| **AI**   | OpenAI Python SDK       | 用于在第三关实现 AI 投资教练功能。                           |
| **前端** | HTML5, CSS3, JavaScript | 负责用户界面和交互。                                         |
| **图表** | Chart.js                | 用于在第二、三关渲染股价历史和投资组合价值曲线。             |

## 核心功能 (Key Features)

### 关卡设计

| **关卡** | **名称** | **目标与核心机制**                                           | **对应的 Python 文件**     |
| -------- | -------- | ------------------------------------------------------------ | -------------------------- |
| **1**    | 银行储蓄 | 通过选择不同的**存款期限和年化利率**，学习**复利**的概念，使本金达到目标 **$10,400**。 | `level_one_banking.py`     |
| **2**    | 初识股票 | 专注于一支科技股 `TECH_A`，学习**低买高卖**的基本股票交易，使总资产达到目标 **$10,800**。 | `level_two_stock.py`       |
| **3**    | 投资组合 | 管理**多支股票**和**现金流**，并处理特殊**市场事件（Missions）**，学习**多元化投资**和**风险管理**。 | `level_three_portfolio.py` |
